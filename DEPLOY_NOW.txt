┌───────────────────────────────────────────────────────────────┐
│ 步骤1：部署到Vercel（3分钟）                                    │
└───────────────────────────────────────────────────────────────┘

  在终端执行：

    cd /Users/dongting/Projects/LevinStreaming/tmdb-proxy
    npm i -g vercel
    vercel login
    vercel --prod

  部署完成后，记录URL：
    
    ✅ Production: https://tmdb-proxy-XXXXX.vercel.app
                   ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                   复制这个URL！


┌───────────────────────────────────────────────────────────────┐
│ 步骤2：配置App（1分钟）                                         │
└───────────────────────────────────────────────────────────────┘

  打开文件：

    LevinStreaming/Core/Utilities/Constants.swift

  找到第20行，修改为：

    static let tmdbProxyURL = "https://tmdb-proxy-XXXXX.vercel.app/api"
                              ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                              替换为你的URL + /api


┌───────────────────────────────────────────────────────────────┐
│ 步骤3：测试（1分钟）                                            │
└───────────────────────────────────────────────────────────────┘

  1. 测试代理（浏览器）：
  
     https://tmdb-proxy-XXXXX.vercel.app/api/trending/movie/week?language=en-US&page=1
  
     应该看到JSON数据 ✅

  2. 运行App（Xcode）：
  
     Cmd + Shift + K (Clean)
     Cmd + R (Run)

  3. 检查日志：
  
     🌐 Using TMDB proxy: https://...
     📡 Network request: https://...
     ✅ Response received in 0.85s


┌───────────────────────────────────────────────────────────────┐

  ⚠️  tmdbProxyURL 必须包含 /api 后缀！
  
      ❌ 错误：https://tmdb-proxy-XXXXX.vercel.app
      ✅ 正确：https://tmdb-proxy-XXXXX.vercel.app/api
